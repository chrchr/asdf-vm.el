# asdf-vm.el

[asdf-vm](https://asdf-vm.com) integration for Emacs

# Synopsis

```emacs-lisp
(require 'asdf-vm)
(asdf-vm-init)
```

# Installation

<kbd>M-x</kbd> <code>package-install</code> <kbd>Return</kbd> <code>asdf-vm</code>
